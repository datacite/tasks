# DataCite

DataCite is a leading global non-profit organisation that provides persistent identifiers (DOIs) for research data. Our goal is to help the research community locate, identify, and cite research data with confidence.

### About this repository


This is the DataCite repository for tasks and other issues.

> [!WARNING]
> This repository is archived. If you have product feedback, visit the [DataCite Roadmap](https://datacite.org/roadmap.html) to learn how to contribute.
